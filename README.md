# EDP_Project — Elevator Simulator

A Windows Forms elevator simulator built with Visual Basic .NET

---

## Requirements
- Visual Studio 2019 or later
- .NET Framework 4.8

---

## How to Run

1. Click the green **"Code"** button → **"Download ZIP"**
2. Find the ZIP in your Downloads folder
3. **Right-click ZIP** → **Properties** → check **"Unblock"** → click **OK**
4. Extract the ZIP
5. Open the extracted folder → double-click **`ElevatorSim2.slnx`**
6. Press **`Ctrl + Shift + B`** to build the project
7. Press **`F5`** to run

---

## Files
- `Form1.vb` — Main UI form
- `ElevatorLogic.vb` — Elevator logic and controls
